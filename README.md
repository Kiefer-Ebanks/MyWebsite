# Personal Website

A modern, responsive personal website featuring a home page, projects section, hobbies section, and resume section.

## Features

- **Responsive Design**: Works perfectly on desktop, tablet, and mobile devices
- **Modern UI**: Clean, professional design with smooth animations
- **Interactive Elements**: Hover effects, smooth scrolling, and fade-in animations
- **Mobile Navigation**: Hamburger menu for mobile devices
- **Smooth Scrolling**: Seamless navigation between sections
- **Loading Animation**: Professional loading screen
- **Typography**: Beautiful Inter font for excellent readability

## Sections

### 1. Home Page
- Hero section with introduction
- Call-to-action buttons
- Social media links
- Professional profile area

### 2. Projects Section
- Project cards with descriptions
- Technology tags
- Links to code and live demos
- Hover animations

### 3. Hobbies Section
- Visual hobby cards with icons
- Descriptions of personal interests
- Interactive hover effects

### 4. Resume Section
- Downloadable resume option
- Professional experience timeline
- Education history
- Skills categorization
- Sticky download card

## Customization Guide

### 1. Personal Information
Replace the following placeholders in `index.html`:
- "Your Name" - Replace with your actual name
- Social media links - Update with your profiles
- Email address - Add your contact email
- Professional title - Update the subtitle

### 2. Projects
Edit the projects in the projects section:
- Update project titles and descriptions
- Modify technology tags
- Add your actual project links
- Replace placeholder icons with relevant ones

### 3. Hobbies
Customize the hobbies section:
- Update hobby titles and descriptions
- Change icons to match your interests
- Add or remove hobby cards as needed

### 4. Resume
Update the resume section:
- Add your actual work experience
- Update education information
- Modify skills lists
- Link to your actual resume PDF

### 5. Styling
Customize the appearance in `styles.css`:
- Change color scheme by modifying CSS variables
- Update fonts by changing the Google Fonts import
- Adjust spacing and layout as needed

### 6. Colors
The current color scheme uses:
- Primary: Blue (#2563eb)
- Secondary: Yellow (#fbbf24)
- Background: Light gray (#f9fafb)
- Text: Dark gray (#1f2937)
